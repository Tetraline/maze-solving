# Maze Solving Algorithm

## Problem Statement 

Given a maze made up of wall and path, find a route between the entrance and the exit. 

## Technologies Used
- Java
- Swing toolkit for the GUI
- JavaDoc
- JUnit testing

## Deployment
[Live Demo on Replit](https://replit.com/@Tetraline/maze-solving)
